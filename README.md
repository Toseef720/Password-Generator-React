# 🔐 Random Password Generator (React)

A modern and responsive web application that generates secure and customizable random passwords. Built using **React** and styled with **Tailwind CSS**.

---

## 🚀 Features

* 🔑 Generate strong random passwords instantly
* 🎛️ Adjustable password length
* 🔤 Toggle uppercase letters
* 🔡 Toggle lowercase letters
* 🔢 Include numbers
* 🔣 Include special characters
* 📋 Copy to clipboard with one click
* 📱 Fully responsive design
* ⚡ Fast and lightweight (client-side only)

---

## 🛠️ Tech Stack

* **React.js** – Frontend framework
* **Tailwind CSS** – Styling
* **JavaScript (ES6+)** – Logic

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```id="n7x2c5"
git clone https://github.com/Toseef720/Password-Generator-React.git
```

### 2️⃣ Navigate to the project directory

```id="p4m9v1"
cd password-generator-react
```

### 3️⃣ Install dependencies

```id="w8k6q3"

```

### 4️⃣ Start the development server

```id="y2r5z7"

```
